# Analysis-of-Breast-cancer-data
This project presents an Exploratory Data Analysis (EDA) of the Breast Cancer Wisconsin Dataset using Python, NumPy, Pandas, and Matplotlib. The analysis begins with data inspection to understand the dataset's structure, followed by data cleaning by checking for missing values and duplicate records. Descriptive statistics, including the mean, standard deviation, minimum, and maximum values of the radius_mean feature, are computed to summarize the data. Additionally, various visualizations such as bar charts, scatter plots, box plots, and histograms are created to explore feature distributions and the relationship between variables. This project demonstrates fundamental data analysis and visualization techniques that form the foundation for building machine learning models.
